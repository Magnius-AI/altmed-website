# Spanish Translation Plan for AltMed

## Why Spanish Matters
- Manassas: ~20% Hispanic population
- Prince William County: Large Spanish-speaking community
- Competitors likely don't have Spanish sites = competitive advantage

---

## Priority Pages to Translate (Phase 1)

1. **Homepage** - First impression
2. **Services Overview** - What you offer
3. **DOT Physical Page** - High-value service, many Hispanic truckers
4. **Weight Loss Page** - Popular service
5. **Contact/Appointment** - Conversion pages

---

## Implementation Options

### Option A: Hostinger Multi-language (Recommended)
If using Hostinger Website Builder:
1. Go to Settings → Languages
2. Add Spanish (Español)
3. Duplicate pages and translate
4. URL structure: altmedfirst.com/es/

### Option B: Translation Plugin (If WordPress)
- WPML ($99/year) - Professional
- Polylang (Free) - Good starter

### Option C: Google Translate Widget (Quick but Lower Quality)
```html
<!-- Add to header -->
<div id="google_translate_element"></div>
<script>
function googleTranslateElementInit() {
  new google.translate.TranslateElement({
    pageLanguage: 'en',
    includedLanguages: 'es',
    layout: google.translate.TranslateElement.InlineLayout.SIMPLE
  }, 'google_translate_element');
}
</script>
<script src="//translate.google.com/translate_a/element.js?cb=googleTranslateElementInit"></script>
```
